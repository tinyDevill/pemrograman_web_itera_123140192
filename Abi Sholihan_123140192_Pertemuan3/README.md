# Aplikasi Web Manajemen Buku
## Deskripsi
Aplikasi web ini merupakan aplikasi yang dapat digunakan untuk membuat daftar dari buku-buku yang kita miliki, yang telah kita beli, dan sedang kita baca.

## Cara menginstallnya dan menggunakannya
1. Unduh zip dari repository ini.
2. Tempatkan di library yang anda inginkan.
3. Ekstrak file zip yang telah diunduh.
4. Masuk ke folder tersebut hingga folder package.json terlihat.
5. Buka git bash ataupun powershell di folder path tersebut.
6. Lakukan 'npm install' dan 'npm install react-router-dom@6'.
7. Masih di folder path yang sama, lakukan 'npm start'.
8. Buka 'localhost:3000' pada peramban yang anda suka.
9. Aplikasi web siap digunakan.

## Fitur React Utama
Fitur utama yang ada pada aplikasi web ini yaitu
1. Web ini menggunakan localStorage sehingga data yang telah diisikan tidak akan menghilang walaupun pengguna membuka kembali web ini setelah beberapa saat.
2. Web ini menggunakan useState dan useEffect dari modul React dalam menyimpan dan memuat data.
